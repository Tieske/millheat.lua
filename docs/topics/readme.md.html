millheat.lua

Lua interface to the Millheat API. Please checkout the documentation for instructions.

Works with plain LuaSocket/LuaSec, as well as with the Copas scheduler. Supports LuaLogging for log output if available.

Status

Early development, session management works, all basic methods implemented. The newer control endpoints can be used, but do not have their own functions yet.
